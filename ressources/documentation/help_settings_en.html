Assisted-MapMatching
Settings Tab

Here you can modify the settings of the mapmatching process to adjust the algorithms' behaviors to your data.

Stop speed threshold:

Enter the value of the speed threshold under which we will consider that points are not moving.

Topological tolerance

Close call:

Enter the value representing the distance threshold under which two nodes should be connected.

Intersection and Dangle Nodes:

Enter the value representing the distance threshold under which two segments should be connected / are intersecting. This value is needed to avoid errors induced by imprecision in decimals of the geometries' coordinates. The recommanded value is 0.01

Map-matching tolerance

Searching radius:

This value represents the searching radius around every points used by the map-matching function to find a likely path on the street network.

Sigma

This value is used to calculate the probability that a points must be matched on a specific road, based on the distance between the two geometries. The biggest sigma is, the slowest is the decay in probability with distance.
