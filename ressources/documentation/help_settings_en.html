Settings Tab

Here you can tinker the settings inside our function to obtain the expected result

Limit stop speed:

This slider represent the limit where we will consider every point with a speed value inferior to it at the stop status

Topological tolerance

Close call :

This slider represent the length at wich we will consider that two point touch each other and need to be merged

Intersection and Dangle Nodes :

Because of the topological functions that truncate the coordinates of every geometry, we need a tolerance value to identify which points are connected together.

Map-matching tolerance :

Searching radius

This slider is a parameter in the map matching function that represent the searching radius of a possible path around every point

Sigma

This slider is a parameter in the map matching function that represent the probabilities that a road have to be taken considering it's distance to the point. (greater = more likely)
