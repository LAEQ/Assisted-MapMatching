Q3M User Friendly Map matching
Settings Tab

Here you can tinker the settings inside our function to obtain the expected result

Limit stop speed:

This slider represent the limit where we will consider every point with a speed value inferior to it at the stop status

Topological tolerance

Close call :

This slider represent the length at wich we will consider that two point touch each other and need to be merged

Intersection and Dangle Nodes :

This slider represent the length at wich we will consider that two point touch each other and we need to cut the lines

Map-matching tolerance :

Searching radius

This slider is a parameter in the map matching function that represent the searching radius around every point of the path

Sigma

This slider is a parameter in the map matching function that represent the probabilities that a road have to be taken considering it's distance to the point. (greater = more likely)

Here is a few tips that could be the reason why the plugin doesn't work:

- Your map layer isn't complete and the algorithm couldn't find a connection between two part of the path

- The map matching parameter search radius is too low or too high
